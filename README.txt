CONSIGLI PER L'ESECUZIONE DEL PROGRAMMA 

questo è il progetto completo da cli, quello GUI è in sviluppo. Ho pensato di consegnare questo essendo che è completo.

Per l'accesso al database c'è una fase di login; username=ADMIN e password=CIAO e viene fatto un controllo nel database dove c'è una tabella con gli utenti salvati. Le password nella db non vengono salvati in modo chiaro ma un valore di hash che viene calcolato tramite un file allegato insieme. 

Ogni volta che un elemento all'interno del db viene eliminato invia in automatico una mail , ind. mail del ricevente puo essere cambiato nel fine smtplib_mail.py per testare la funzionalità.

Nel programma c'è una funzionalità aggiuntiva che è quella di salvare in un file, le letture del db per avere un backup dei elementi, si deve cambiare il percorso sorgente a rr. 7 se si vuole eseguire il codice. 	

le mie tabelle sono diverse da quelle della scuola anche il nome del db invece di essere 5ATepsit è 5Atepsit. Le tabelle sono allegate.
